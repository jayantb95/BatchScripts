# BatchScripts
Batch Scripts to automate certain tasks on your windows machine.

<h3>Scripts in this Repository:</h3>
<ul>
    <li> Start Hotspot</li>
    <li> Stop Hotspot</li>
    <li> Delete Hotspot</li>
    <li> Clear <i>Temp</i> Folder</li>
    <li> Clear <i>%Temp%</i> Folder</li>
    <li> Clear <i>Prefetch</i> Folder</li>
</ul>
